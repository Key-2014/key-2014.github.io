# Key's Sandbox

This repository contains the source code and deployment data for my personal website, [Key's Sandbox](https://Key-2014.github.io/).
The site is built with [Hugo](https://gohugo.io/), a fast and modern static site generator, and uses the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

- [Site Architecture Guide](https://Key-2014.github.io/post/site-architecture-guide/)
- [Content Editing Guide](https://Key-2014.github.io/post/site-editing-guide/)

## 🛠️ Local Development

To preview the site locally, ensure you have Hugo installed on your system. Then, follow these steps:

```bash
# Clone the repository including submodules (required for the theme)
git clone --recursive https://github.com/Key-2014/Key-2014.github.io.git
cd Key-2014.github.io

# Start the local development server (includes drafts)
hugo server -D
```
